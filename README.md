# hsgPy

This is the working version of the Horizontal Steerable Grating calibration package. It is in very early development state at current. HSG is a spectrograph at the Dunn. It currently uses a mashup of the FIRS and SPINOR reduction scripts. This package aims to duplicate and improve that functionality in python for merge with SSOSoft. As such, it will eventually have a class structure and dedicated config file similar to the existing package routines.

At current moment, all that's here is the low level tools for characterizing and working with spectra. These should be universal for HSG, FIRS, and SPINOR. I'm including them as a separate function file for eventual use with all of them.
